== Description ==

My Prayer Time is a WordPress plugin made by Xhanch Studio to display Moslem prayer time table based on visitor's IP (daily and monthly).

It provide a widget to display a daily prayer time, a shortcode to be placed on a page to display a monthly prayer time and a print version monthly prayer time.
